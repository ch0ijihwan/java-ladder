# java-ladder
## 요구 사항
- 사다리 게임에 참여하는 사람 이름은 최대 5글자 까지 부여할 수 있다.
- 사람 이름은 쉼표(,)를 기준으로 구분한다.
- 사람 이름을 5자 기준으로 출력하기 때문에 사다리 폭도 넓어져야한다 → 사다리 폭은 5자 이상
- 사다리 타기가 정상적으로 동작하려면 라인이 겹치지 않도록 해야한다. → 한 라인에 모든 구간에 사다리가 있으면 안됨

- 사용자로부터 참여할 플레이어의 이름을 입력 받는다.
    - 이름은 최대 5자이며, ( , ) 을 기준으로 자른다.
    - 인원수를 반환해야한다.
- 최대 사다리 높이를 입력 받는다.
    - 사다리의 높이는 자연수어야 한다
- 사다리를 생성한다.
    - 생성된 사다리의 라인이 겹치면 안된다.
    - 사다리의 좌표(세로 라인)는 인원 수 만큼 갖고 있어야한다.
    - 사다리 라인(가로 라인) 으로 이루어진 사다리를 갖는다.
